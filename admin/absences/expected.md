### Expected Absence
If you know you're going to be away for a session and you're one of the [Core Group](https://github.com/millydrisco/tod-docs/blob/main/README.md#core-group), you should have access to the google calendar - update it. This links to updating the discord bot that posts every Monday. 

**Take the following steps:**
1. Try to have a conversation in discord to rearrange the session so we can still all play. *It is up to you to then change the session date in the Google Cal, please, so it reflects the correct day.*
2. If a new day can't be arranged - let Mattie know! It's on them to sort Vecna shit out, none of my business.

#### Caveats
* An expected absence is one with like 24hrs notice or more. We can all pretty much sort stuff out with 24hrs notice.
* If the expected absence is due to something emotional, or otherwise impacting that would mean the above steps are too much - reach out, I don't mind doing the work.
* These steps also apply to me.
